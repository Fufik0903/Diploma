Командная строка

Существует множество различных способов использования Git. Существуют оригинальные инструменты командной строки и множество графических пользовательских интерфейсов с различными возможностями. Для этой книги мы будем использовать Git в командной строке. Во-первых, командная строка-это единственное место, где вы можете запустить все Команды Git — большинство GUIs реализуют только частичное подмножество функций Git для простоты. Если вы знаете, как запустить версию командной строки, вы, вероятно, также можете выяснить, как запустить версию GUI, в то время как противоположное не обязательно верно. Кроме того, в то время как ваш выбор графического клиента является делом личного вкуса, все пользователи будут иметь установленные и доступные инструменты командной строки.
